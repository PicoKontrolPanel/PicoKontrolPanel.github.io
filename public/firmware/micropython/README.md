# Bundled MicroPython UF2

`RPI_PICO_W.uf2` is the bundled MicroPython image used by Kodeværksted's guided installer.

- Board: Raspberry Pi Pico W
- MicroPython: v1.28.0
- Release date: 2026-04-06
- Source: https://micropython.org/download/RPI_PICO_W/

To update it, download the latest stable `.uf2` for Pico W from MicroPython and replace `RPI_PICO_W.uf2`.
